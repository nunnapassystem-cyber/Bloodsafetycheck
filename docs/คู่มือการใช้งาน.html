ส่วนที่ 1

คู่มือสำหรับพยาบาล (User)

ขั้นตอนการใช้งานระบบตรวจสอบความปลอดภัยก่อนให้เลือดผู้ป่วย

1. การเข้าสู่ระบบ

  1. เปิด Browser บนมือถือหรือ Tablet → ไปที่ URL ของระบบ
  2. กรอก Email และ Password ที่ได้รับจาก Admin
  3. กด "เข้าสู่ระบบ"
  4. หน้าหลักจะแสดง ชื่อพยาบาล และ Ward ของท่านที่มุมบนขวา
ℹ️
ลืมรหัสผ่าน

ติดต่อผู้ดูแลระบบ (Admin) เพื่อ Reset รหัสผ่าน

2. ขั้นตอนที่ 1 — Scan และระบุข้อมูล

1
Scan สติ๊กเกอร์ผู้ป่วย (จากแบบบันทึกการให้โลหิต)
  1. กดปุ่ม "Scan ชื่อผู้ป่วย หน้า Order สั่งให้เลือด"
  2. ให้กล้องจับ Barcode/QR Code บนแบบบันทึกการให้โลหิต
  3. ระบบอ่าน HN อัตโนมัติ
⚠️
กรณี Scan ไม่ได้

กด "Scan ใหม่" เพื่อลองอีกครั้ง หรือกด "พิมพ์ HN เอง" → กรอก HN ด้วยมือ → กด ยืนยัน

2
Scan ถุงเลือด
  1. หลังได้ HN แล้ว → กดปุ่ม "Scan ถุงเลือด"
  2. ให้กล้องจับ Barcode บนถุงเลือด
  3. ระบบบันทึกรหัสถุงเลือดอัตโนมัติ
3
กรอกข้อมูลตรวจสอบ

กรอกข้อมูลให้ครบทุกช่อง:

ช่องข้อมูลอ่านจาก
ชื่อ-สกุล ผู้ป่วยแบบบันทึกการให้โลหิต
Blood Group ผู้ป่วย (ABO + Rh)แบบบันทึกการให้โลหิต
ชนิดเลือดที่สั่ง (Order แพทย์)แบบบันทึกการให้โลหิต
Blood Group ถุงเลือด (ABO + Rh)ป้ายถุงเลือด
ชนิดเลือดในถุงป้ายถุงเลือด
ปริมาณ (ml)ป้ายถุงเลือด

จากนั้นกด "ยืนยัน → ตรวจสอบ Blood Group + ชนิดเลือด"

3. ผลการตรวจสอบ

✅ PASS — ข้อมูลตรงกัน
  • ระบบแสดงสรุปสีเขียว
  • ไปขั้นตอนที่ 2 ได้เลย
❌ FAIL — ข้อมูลไม่ตรง
  • Alert สีแดง + เสียงเตือน
  • ห้ามให้เลือดโดยเด็ดขาด
  • ระบบบันทึก FAIL log อัตโนมัติ
  • ส่งถุงเลือดคืน Blood Bank
  • กด "ล้างข้อมูล เริ่มใหม่"

4. ขั้นตอนที่ 2 — ยืนยันด้วยพยาบาล 2 คน

  1. ตรวจสอบสรุปข้อมูลที่แสดงบนหน้าจอ
  2. พยาบาลคนที่ 1 — ชื่อแสดงอัตโนมัติจากการ Login (ไม่ต้องกรอก)
  3. พยาบาลคนที่ 2 — กรอกชื่อพยาบาลที่ยืนยันร่วม (ต้องเป็นคนละคนกับพยาบาลที่ 1)
  4. กด "ยืนยันเริ่มให้เลือด — บันทึกเวลา"
🚫
ห้ามกรอกชื่อตัวเองซ้ำ

ระบบตรวจจับและไม่อนุญาตให้พยาบาลคนที่ 1 และ 2 เป็นชื่อเดียวกัน

5. ขั้นตอนที่ 3 — หน้าสรุปสำเร็จ

หลังยืนยัน ระบบแสดง ✅ บันทึกสำเร็จ พร้อมสรุปข้อมูลการให้เลือดทั้งหมด

การบันทึกภาพ / แชร์ไป LINE:

  1. กด "บันทึกภาพ / แชร์ไป LINE"
  2. มือถือเปิด Share sheet → เลือก LINE หรือ บันทึกรูป
  3. ส่งเข้า Line กลุ่ม Ward ได้เลย

เริ่มผู้ป่วยรายต่อไป:

กด "เริ่มผู้ป่วยรายต่อไป" → ระบบล้างข้อมูลและกลับสู่ขั้นตอนที่ 1 อัตโนมัติ

6. Session Timeout และการออกจากระบบ

⏱ Session Timeout
  • ไม่ใช้งาน 5 นาที → ล้างข้อมูลผู้ป่วยอัตโนมัติ
  • ต้องเริ่มขั้นตอนที่ 1 ใหม่
  • ไม่ต้อง Login ใหม่
🚪 ออกจากระบบ
  • กดปุ่ม Logout มุมบนขวา
  • ควร Logout ทุกครั้งที่ใช้งานเสร็จ

คู่มือสำหรับผู้ดูแลระบบ (Admin)

การจัดการ Audit Log, ผู้ใช้งาน และการดูแลระบบ

7. การเข้าสู่ระบบ Admin

  1. Login ด้วย Email/Password Admin ที่ตั้งค่าไว้
  2. หลัง Login จะเห็น Admin Dashboard ซึ่งแตกต่างจากหน้าพยาบาล
  3. Dashboard แสดง Audit Log จากทุก Ward

8. การดู Audit Log

คอลัมน์ที่แสดงในตาราง:

คอลัมน์ความหมาย
Wardชื่อหอผู้ป่วย
เวลาวันเวลาที่ให้เลือด
Wristband IDรหัส Barcode ป้ายข้อมือผู้ป่วย
ถุงเลือดรหัสถุงเลือด
ชนิดPRC / FFP / Platelet / WB
Blood Groupหมู่โลหิตถุงเลือด
ผลการ Match✅ PASS หรือ ❌ FAIL
พยาบาล 1 / 2ชื่อพยาบาลที่ยืนยัน

การ Filter:

กรองตามวันที่
  • เลือกวันที่ต้องการดู
  • แสดงเฉพาะ Log ของวันนั้น
กรองตามผล
  • PASS — แสดงเฉพาะที่ผ่าน
  • FAIL — แสดงเฉพาะที่ไม่ผ่าน
  • ทั้งหมด — แสดงทุกรายการ
📥
Export ข้อมูล CSV

Supabase Dashboard → Table Editor → transfusion_logs → Export → CSV

9. การจัดการผู้ใช้งาน

เพิ่มพยาบาล / User ใหม่

  1. เข้า Supabase Dashboard → Project → Authentication → Users
  2. กด "Add User → Create new user"
  3. กรอก Email และ Password
  4. หลัง Create → คลิกที่ User → "Edit User Metadata"
  5. กรอก metadata ดังนี้:
{ "nurse_name": "ชื่อ-สกุล พยาบาล", "ward_id": "ward-A", "ward_name": "ศัลยกรรมชาย", "role": "nurse" }

ตาราง ward_id

ward_idward_nameward_idward_name
ward-Aศัลยกรรมชายward-Hวิกฤตอายุรกรรม
ward-Bศัลยกรรมหญิงward-Iวิกฤตอุบัติเหตุ, ศัลยกรรมหัวใจ
ward-Cอายุรกรรมชายward-Jห้องผ่าตัดศัลยกรรม
ward-Dกุมารเวชกรรมward-Kห้องผ่าตัดสูติกรรม
ward-Eสูติกรรมward-Lอุบัติเหตุชาย
ward-Fอายุรกรรมหญิงward-Mอาคาร 1 ชั้น 4
ward-Gวิกฤตศัลยกรรมward-Nอาคาร 2 ชั้น 6

เพิ่ม Admin ใหม่

{ "nurse_name": "ชื่อ-สกุล ผู้ดูแลระบบ", "ward_id": "admin", "ward_name": "ผู้ดูแลระบบ", "role": "admin" }
🔑 Reset รหัสผ่าน
  • Supabase → Authentication → Users
  • คลิก User → Send Password Reset Email
  • ระบบส่ง link ไปที่ Email
🗑 ลบ User
  • Supabase → Authentication → Users
  • คลิก User ที่ต้องการลบ
  • กด Delete User

10. นโยบายข้อมูลและความปลอดภัย

หัวข้อรายละเอียด
ระยะเวลาเก็บ Log60 วัน — ลบอัตโนมัติทุกวันตี 4 น. (Vercel Cron)
