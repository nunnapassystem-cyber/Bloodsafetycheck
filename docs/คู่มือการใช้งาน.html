บันทึกสำเร็จ

1. การเข้าสู่ระบบ

  1. เปิด Browser บนมือถือหรือ Tablet → ไปที่ URL ของระบบ
  2. เลือก Ward ของท่านจากรายการ (แสดงเฉพาะ Ward ที่ Admin เปิดใช้งาน)
  3. เลือก ชื่อพยาบาล ของท่าน
  4. กรอก Password ที่ได้รับจากผู้ดูแลระบบ
  5. กด "เข้าสู่ระบบ"
ℹ️
ลืมรหัสผ่าน

กด "เปลี่ยนรหัสผ่าน" บนหน้าหลัก หรือติดต่อ Admin เพื่อ Reset

2. ขั้นตอนที่ 1 — ตรวจ Order + ถุงเลือด

มี 4 หัวข้อย่อย (ก–ง) ทำตามลำดับทุกครั้ง

ก ถ่ายรูปสติ๊กเกอร์ข้อมือ
  1. กด "ถ่ายรูปสติ๊กเกอร์ข้อมือ" → กล้องเปิด
  2. ถ่ายรูปสติ๊กเกอร์ที่ข้อมือผู้ป่วย
  3. ปรับกรอบตัดให้ครอบข้อความ → กด "ตกลง อ่านข้อความ"
  4. ระบบอ่าน HN และชื่อผู้ป่วยอัตโนมัติ — แสดงในกล่องสีน้ำเงิน
  5. แก้ไขได้โดยตรงถ้า OCR อ่านผิด
⚠️
OCR อ่านไม่ได้

กด "กรอก HN เอง" → พิมพ์ HN → กด "ยืนยัน HN"

ข กรอกข้อมูลจากแบบบันทึกการให้โลหิต
ช่องอ่านจาก
Blood Group ผู้ป่วย (ABO)แบบบันทึกการให้โลหิต
Rh ผู้ป่วย (Positive/Negative)แบบบันทึกการให้โลหิต
ชนิดเลือดที่สั่ง (Order แพทย์)แบบบันทึกการให้โลหิต

ระบบแสดง Blood Group สรุป เช่น A+ หรือ O- หลังเลือกครบ

ค ถ่ายรูปบัตรคล้องถุงเลือด
  1. กด "ถ่ายรูปบัตรคล้องถุงเลือด" → ถ่ายบัตรสีขาวที่คล้องถุงเลือด
  2. ปรับกรอบตัดให้ครอบข้อความ → กด "ตกลง อ่านข้อความ"
  3. ระบบอ่านข้อมูลถุงเลือดอัตโนมัติ (แก้ไขได้ทุกช่อง):
  • ชื่อ/สกุล และ HN บนถุงเลือด
  • ชนิดเลือด (PRC / LPRC / FFP / Platelet / WB)
  • Blood Group, Rh, หมายเลขถุงเลือด, ปริมาณ (mL)
ง ตรวจสอบข้อมูลที่ตรงกัน
รายการผล
✅ ชื่อ-สกุล ตรงกันดำเนินการต่อได้
⚠️ ชื่อ-สกุล ต่างกันตรวจสอบอีกครั้ง (OCR อาจอ่านผิด)
✅ HN ตรงกันดำเนินการต่อได้
❌ HN ไม่ตรงกันห้ามให้เลือด — ระบบบล็อก

เมื่อข้อมูลตรงกัน → กด "ยืนยัน → ตรวจสอบ Blood Group + ชนิดเลือด"

3. ผลการตรวจสอบ Blood Group

✅ PASS — ข้อมูลตรงกัน
  • ระบบแสดง Checklist ก่อนออกไปเตียง:
  • ☑ เตรียมป้ายมือให้เลือด
  • ☑ เตรียมใบยินยอมให้เลือด
  • กดทั้ง 2 ข้อ → กด "ไปขั้นตอน 2"
❌ FAIL — ข้อมูลไม่ตรง
  • แถบแดง + เสียงเตือนดัง
  • แสดงสาเหตุชัดเจน เช่น:
    "Blood Group ไม่ตรง: A+ / B+"
  • ห้ามให้เลือดโดยเด็ดขาด
  • ระบบบันทึก FAIL log อัตโนมัติ
  • กด "ล้างข้อมูล เริ่มใหม่"

4. ขั้นตอนที่ 2 — ยืนยันด้วยพยาบาล 2 คน

  1. ตรวจสอบสรุปข้อมูลทั้งหมดอีกครั้ง
  2. พยาบาลคนที่ 1 — แสดงชื่ออัตโนมัติจาก Login (ไม่ต้องกรอก)
  3. พยาบาลคนที่ 2 — กรอกชื่อพยาบาลที่มายืนยันร่วม (ต้องเป็นคนละคน)
  4. กด "ยืนยันเริ่มให้เลือด — บันทึกเวลา"

5. ขั้นตอนที่ 3 — บันทึกสำเร็จ

ระบบแสดง ✅ บันทึกสำเร็จ พร้อมเวลาเริ่มให้เลือดและสรุปข้อมูลทั้งหมด

ปุ่มทำอะไร
บันทึกภาพ / แชร์ไป LINEสร้างภาพสรุป → แชร์เข้า LINE กลุ่ม Ward
ถุงเลือดถัดไปสแกนถุงเลือดใบใหม่ (ผู้ป่วยคนเดิม)
ผู้ป่วยรายต่อไปล้างข้อมูลทั้งหมด เริ่มต้นใหม่

6. Session Timeout & เคล็ดลับ OCR

⏱ Session Timeout
  • ไม่ใช้งาน 5 นาที → ล้างข้อมูลผู้ป่วยอัตโนมัติ (PDPA)
  • ต้องเริ่มขั้นตอนที่ 1 ใหม่
  • ไม่ต้อง Login ใหม่
📷 เคล็ดลับถ่ายรูปให้ OCR แม่น
  • ถ่ายในที่มีแสงเพียงพอ หลีกเลี่ยงเงา
  • ถือมือถือนิ่ง ไม่เบลอ
  • ปรับกรอบตัดครอบเฉพาะข้อความ
  • OCR อ่านผิด → แก้ไขในช่องได้เลย

ส่วนที่ 2

คู่มือสำหรับผู้ดูแลระบบ (Admin)

การจัดการผู้ใช้งาน, Ward, Audit Log และการดูแลระบบ

7. การเข้าสู่ระบบ Admin

  1. หน้า Login → เลือก Ward: ผู้ดูแลระบบ → เลือกชื่อ Admin → กรอก Password
  2. หลัง Login จะเห็นเมนูเพิ่มเติม: Admin · Audit Log · เปลี่ยนรหัสผ่าน

8. การดู Audit Log & OCR Quota

ไปที่เมนู Audit Log — Admin เห็น Log ทุก Ward, พยาบาลเห็นเฉพาะ Ward ตัวเอง

ฟีเจอร์การกรอง:

กรองตามวันที่
  • เลือกวันที่ → แสดงเฉพาะ Log วันนั้น
กรองตามผล
  • PASS · FAIL · ทั้งหมด
  • Export Excel (.xlsx)

ยอด OCR Scans เดือนนี้ (แสดงด้านบนตาราง):

OCR Scans เดือนนี้ 245 / 1,000
⚠️ OCR Scans เดือนนี้ — ใกล้ถึงลิมิต 920 / 1,000

เมื่อครบ 900 scan ระบบหยุด OCR อัตโนมัติ — พยาบาลต้องกรอก HN เอง (Quota reset ทุกวันที่ 1 ของเดือน)

9. เพิ่ม Nurse User ใหม่

ไปที่ Admin → ส่วนที่ 1 "เพิ่ม Nurse User ใหม่"

  1. กรอก Email — อีเมลพยาบาล (ใช้ Login)
  2. กรอก ชื่อ-สกุล พยาบาล
  3. เลือก Ward ที่พยาบาลประจำ
  4. กรอก Password เริ่มต้น (อย่างน้อย 6 ตัวอักษร)
  5. เลือก Role: Nurse หรือ Admin
  6. กด "สร้าง User" — ระบบสร้างบัญชีทันที
ℹ️

Admin 1 Ward มีได้สูงสุด 2 คน — ระบบแจ้งเตือนถ้าเต็ม

10. เปิด/ปิด Ward ที่แสดงในหน้า Login

ไปที่ Admin → ส่วนที่ 2 "สถานะ Ward (แสดงในหน้า Login)"

แต่ละ Ward Card มี Toggle Switch ที่มุมบนขวา:

เปิด (สีน้ำเงิน) — Ward ปรากฏในหน้า Login ให้พยาบาลเลือกได้
ปิด (สีเทา) — Ward ถูกซ่อน พยาบาลไม่เห็นในหน้า Login
💡
วิธีใช้

กด Toggle Switch → ระบบบันทึกทันที ไม่ต้อง Refresh หน้า Login
Ward ที่ปิดยังสามารถสร้าง/แก้ไข Nurse ได้ตามปกติ เพียงแต่ไม่แสดงในหน้า Login

11. แก้ไข / ลบ Nurse User

ไปที่ Admin → ส่วนที่ 3 "รายชื่อ Nurse ทั้งหมด"

แก้ไขข้อมูล:

  1. กด "แก้ไข" ข้างชื่อพยาบาล
  2. แก้ไขชื่อ / Ward / Role / Password ได้ตามต้องการ
  3. กด "บันทึก" หรือ "ยกเลิก"

ลบ Nurse (กรณีลาออก ฯลฯ):

  1. กด "แก้ไข" → เลื่อนดูด้านล่างสุด
  2. กด "ลบชื่อออกจากระบบ" (ข้อความสีแดง)
  3. ระบบขอยืนยัน: "ยืนยันลบ [ชื่อพยาบาล] ?"
  4. กด "ลบเลย" เพื่อยืนยัน หรือ "ยกเลิก"
🚫
ข้อควรระวัง

Admin ไม่สามารถลบบัญชีตัวเองได้ — ระบบป้องกันอัตโนมัติ

Reset รหัสผ่านพยาบาล:

วิธีที่ 1 — Admin Panel
  • Admin → รายชื่อ Nurse → แก้ไข
  • กรอก Password ใหม่ → บันทึก
วิธีที่ 2 — Supabase
  • Authentication → Users → คลิก User
  • Send Password Reset Email
